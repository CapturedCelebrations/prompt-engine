# AI Photo Booth Prompt Engine

Generate expert-level AI prompts for photo booth AI filters. Built by Captured Celebrations.

## Features

- **Build Mode** — Step-by-step guided prompt builder with auto filter selection
- **Reverse Engineer** — Upload any AI photo booth image, get the exact prompt + settings to recreate it
- 11 art styles: Cinematic, Watercolor, Caricature, Pixar 3D, Anime, Fantasy, Retro, Luxury, B&W, Collectible Box, Custom
- 9 event themes: Wedding, Corporate, Quinceañera, Birthday, Holiday, Cultural, Fashion, Sports, Custom
- Smart filter recommendation based on your setup
- Negative prompt generation
- Complete booth setup checklist
